Introduction

Knowledge Base

The knowledge base is a set of howto,articles, reference and technical documents to assist the user/developer

Start by clicking on the Create button below and then select a simple or complex template

Once the template has loaded enable the inline html editor by clicking on Editor

The updated document can then be saved in the desired category directory Save

Rename the categories as desired - remember that all directories and file names are case sensitive

 

Todo List

A day based todo list (based on the current day saved using yyyymmdd format)

Click on the Todo List menu item

Start by clicking on Add New Task button

Enter the item with a title, status and a label all sepereated by a ','

Status can be anything really - usually time related now,today,tomorrow, 2 days etc

Labels are on of [danger,success,info,primary,warning and inverse]

Click on the items on the right of each task to complete or edit
